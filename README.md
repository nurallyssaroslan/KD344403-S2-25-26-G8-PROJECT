# KD344403-S2-25-26-G8-PROJECT

# Dataset Used
Global AI vs Human Content Dataset 2026 by Asif Zaman on Kaggle.
The dataset contains 20,000 samples of text and code labeled as either AI-generated or human-written. The data spans multiple domains, including health, finance, education, politics, and technology, providing diverse linguistic patterns for analysis. Each sample includes the main content along with metadata such as prompt, topic, word count, character count, and complexity score. The dataset is balanced between AI and human classes, making it suitable for training and evaluating text classification models.

# Project Objective
To detect human vs AI-generated content
